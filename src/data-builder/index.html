The Tree of Three.js

Data Builder

Data Builder is half-automatic tool that prepares data for the Three of Three.js. It uses data from files data-src.js and data-addons.js found in folder \src\data\. If these files do not exist or are outdated, call REBUILD.BAT from the project root to recreate them.

Data Builder creates JSON file representing a tree of Three.js public classes, functions and variables. This file must be saved as data.js back in the same folder \src\data\. If the box below does not show any text, check the developer's console.
