# The Three Tree

A tree-like visualization of the digital nervous system of Three.js. It includes
all exposed entities - classes, functions, variables and constants. Click on and
image to open it life.

### How to use

Use mouse or touch to orbit around the tree or zoom it in/out the 3D arrangement.

Add URL parameter `show=<string>` to mark all elements with the given string in their
names; `camera=<x>,<y>,<z>` to set initial camera 3D position; `target=<x>,<y>,<z>`
to set initial camera target 3D position.

This URL shows the tree with `node` entities coloured in red and all the rest in white:
```
https://boytchev.github.io/TheThreeTree/src?show=node&camera=-322,317,141&target=-15,247,-55
```
